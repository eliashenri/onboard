## Description

Sistema de cadastro de livros desenvolvido em TypeScript
com a biblioteca Next.js. Utilizei o ORM Sequelize e o banco
de dados MySQL.

## Installation

```bash
 Create a .env file like the ".env.example"
```
```bash
$ npm install
```

## Running the app

```bash
$ npm run start
```
